## ğŸ§  How It Works

Vi3W treats 3D creation as a **reasoning problem**, not a sculpting task.

### Architecture Overview

1. **Gemini 3 Pro (Google AI Studio)**  
   - Multimodal understanding (text + images)  
   - Interprets user intent  
   - Generates parametric 3D instructions  

2. **Python Geometry Pipeline**  
   - Converts instructions into structured 3D geometry  
   - Optimizes meshes / voxel representations  
   - Enables iterative, part-level editing  

3. **Web Viewer (React + Three.js)**  
   - Real-time 3D visualization  
   - Interactive inspection and refinement  
   - Conversational editing loop  

This setup supports both **generation and iteration**, instead of one-shot outputs.
